# URLs potencialmente duplicadas que devem usar canonical
https://fantasystore.com.br/products?page=1 - Canonical para /products
https://fantasystore.com.br/products?sort=price_asc - Canonical para /products
https://fantasystore.com.br/products?sort=price_desc - Canonical para /products
https://fantasystore.com.br/search?q=minecraft - Canonical para /products

# Para implementar em cada página:
# <link rel="canonical" href="URL_CANÔNICA" />
